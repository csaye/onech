# Onech

A one char function programming language.

## Functions

- \< (decrement pointer)
- \> (increment pointer)
- \- (decrement variable)
- \+ (increment variable)
- \* (double variable)
- \/ (half variable)
- \. (print char at pointer)
- \: (print number at pointer)
- \[ (start if statement)
- \] (end if statement)
